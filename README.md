# 🏦 Kod Bank Application

An Online Banking Application developed using Java Full Stack Technology with secure transaction management and modern user interface.

---

## 🚀 Project Overview

Kod Bank App is a full-stack banking management system that allows users to perform secure banking operations digitally.  
The application focuses on user-friendly banking services with real-time transaction handling.

---

## ✨ Features

- 🔐 User Registration & Secure Login
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 🔁 Fund Transfer
- 🧾 Transaction History
- 👤 Profile Management
- 🛡️ Secure Session Handling
